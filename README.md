# 어드민 서비스 

게시판 서비스의 관리 도구를 개발해보는 프로젝트입니다.

## 개발 환경

* Intellij IDEA Ultimate
* Java 17
* Gradle 8.14.2
* Spring Boot 3.5.3

## 기술 세부 스택

Spring Boot

* Spring Boot Actuator
* Spring Web
* Spring Data JPA
* Thymeleaf
* Spring Security
* Spring Security OAuth 2 Client
* H2 Database
* MySQL Driver
* Lombok
* Spring Boot DevTools
* Spring Configuration Processor

그 외

* AdminLTE 3.2
